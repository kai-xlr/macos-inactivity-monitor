# UI Specification

## Dashboard

Displays:
- Running apps
- Last active time
- Status (Active / Inactive)

## Allowlist Manager

Features:
- Add app
- Remove app
- List allowlisted apps

## Settings

- Inactivity threshold (minutes)
- Toggle monitoring

## Alert Popup

Message:
"This app has been inactive for 30 minutes."

Buttons:
- Close App
- Force Quit
- Ignore
