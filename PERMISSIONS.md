# Permissions

## Accessibility Permission (Required)

Used for:
- Detecting user interaction
- Tracking app focus

### How to Enable
System Settings → Privacy & Security → Accessibility

## Optional Permissions

### Full Disk Access
Only needed if:
- Logging deeply
- Inspecting app data

## UX Requirement

The app must:
- Explain clearly why permission is needed
- Fail gracefully if not granted
